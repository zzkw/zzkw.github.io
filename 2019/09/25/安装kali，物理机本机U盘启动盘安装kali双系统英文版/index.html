安装kali，物理机本机U盘启动盘安装kali双系统英文版
发表于 2019-09-25| 更新于 2019-09-25|-Linux -kali
字数总计: 840|阅读时长: 2 分钟|阅读量:

﻿填坑

首先，网上kali官网洪流比特下载好，安装网络采用桥接模式，若上网不了，更改虚拟网络编辑器把IP设为主机前两位相同，后一位不同，eg:192.168.10.0，kali网络设置里改为自动获取ipv4。输入法参考了大佬u013526533的文章，顺序为
leafpad /etc/apt/sources.list
添加deb http://mirrors.ustc.edu.cn/kali-security/ sana/updates main contrib non-free
保存
apt-get update
apt-get install fcitx
apt-get install fcitx-table-wbpy ttf-wqy-microhei ttf-wqy-zenhei
init 6重启即可（老实说这个不好用emmm大家有空还是装个搜狗输入法哈）
安装VMware tools,放到home上在终端tar zxvf VMwareTools-10.0.10-4301679.tar.gz
ls查看目录
输入
./vmware-tools-distrib
./vmware-install.pl
安装完毕。如有错误或问题留言指导我，谢谢点击。

物理机安装更新~本文写些思路，无图、、、耶
本次安装出来很多小问题啊，哎主要是自己装B装着英文版kali，原本U盘老毛桃启动盘不支持linux系统，改成Win32 Disk Imager.这一个就够了真的其他都不要，我装了几个，Universal USB Installer真的装不得，警告哎尤其是挂载镜像的，拔插U盘怀疑人生。

以及Executing ‘grub-install /dev/sda5’ failed. This is a fatal error找不到这路径等等问题太多了，但用Win32 Disk Imager后都消失了，好吧我是装中文版发现没什么问题再装英文版操作一波的哈哈哈哈

首先，肯定是U盘启动，BIOS设置下我这一直都是，easy
进入kali安装界面，选择Graphical Install
选择语言，哈哈哈哈English HongKong 键盘American Key
然后是名称及CLI名（登录名统一root哈）
域名跳过
设置登录密码
分区向导，这里选最后一个（手动）彼此之间不敲英文了（其实我是不熟那怎么拼）
找到你留给kali的空闲磁盘，我给了500G贼刺激，请记住这里给kali空间的盘号就是#1这些后有用的有兴趣分区可看博客https://blog.csdn.net/weixin_39762423/article/details/81119480
点击对磁盘设置，我的是etxt4（第一个）,挂载点选择/根目录（也是第一个）然后Finish（在下面）
诺，图在这里博客https://blog.csdn.net/qq_32923745/article/details/78252441
写入引导（很重要一步）
因为我们的要求是，要使用windows的引导来启动，所以这里我们选择否，如果你想用linux的引导就选是，后面就不用再设置了。continue(继续)
记住选择Enter device manually
之后选择分区编号 我的是/dev/sda1你们的很可能不是这个哈自己看

贴个回答：Executing ‘grub-install /dev/sda5’ failed. This is a fatal error
GRUB是安装在硬盘上，而不是在分区。我假设你想创建一个单独的分区挂载到/ boot，但应该在驱动器上使用GRUB安装（在这种特殊情况下的/ dev / sda上），而不是一个分区。

fdisk -l
要知道的是安装在您的linux
mount /dev/sdaX/ mnt
X是安装您的（分区编号）位置
然后
grub-install –root-directory=/mnt/ /dev/sda
然后重新启动正常，然后
update-grup（我敲这个出错了update-grup2然后还是没搞头选择Win32 Disk Imager写入镜像啊啊啊完事）
ubuntu需要每个命令前加sudo Debian不用
